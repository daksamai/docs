## Setting up


Setting Up

Before you can start integrating the eKYC API into your application, you'll need to sign up for an eKYC account and obtain your API key. Follow these steps to get started:

1) Signup for an Account:
Go to https://demo.ekyc.daksam.ai/sign-up and create your eKYC account.
Verify your email address to activate your account.

2) Login to the Portal:
Once your account is activated, login to the eKYC portal at https://demo.ekyc.daksam.ai/sign-up.


3) Access Developer Space:
In the portal, navigate to the Developer Space menu.
Alternatively, you can go directly to the Developer Space at https://demo.ekyc.daksam.ai/admin/developers.


4)Get Your API Key:
In the Developer Space, you'll find your API key.
Copy your API key and securely store it for use in your integration.
With your API key in hand, you're now ready to integrate the eKYC API into your application and start verifying your users with ease.
